About the Project

This system, "Using Machine Learning To Predict And Monitor Fetal Health", is designed to assist medical professionals and expectant mothers in assessing fetal well-being using intelligent technology. By analyzing clinical cardiotocographic (CTG) data, the model classifies fetal health into categories like normal, suspect, or pathological.

The application leverages machine learning algorithms including Random Forest, Decision Tree, Logistic Regression, and K-Nearest Neighbors to analyze data features. It uses data preprocessing techniques like SMOTE for balancing, feature selection for dimensionality reduction, and standard scaling for model optimization.

Our goal is to support early diagnosis of fetal risks, reduce human error in interpretation, and ultimately enhance prenatal care using reliable, fast, and accurate predictions powered by AI.
